QueryForge

Введи запрос свободно — получи подзапросы для Composer/Agent

Ключ хранится только в браузере (localStorage) и не передаётся на сторонние серверы.

Контекст проекта (опционально)
Облачная версия (Vercel): контекст не подтягивается автоматически. Вставьте контекст вручную по структуре из (см. раздел «Контекст проекта» в README).
Зачем нужен: структура папок, ключевые файлы — подзапросы будут с точными @ путями (@auth/, @services/). Без контекста — разбивка по типичным слоям, пути могут не совпасть с проектом.
Особенно важно для ревью, аудита, поиска багов — чтобы не пропустить модули.
— для ознакомления с примерным результатом, без расхода токенов
